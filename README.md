AI-Powered Energy Management System
An AI-powered energy management system that collects real-time industrial electrical data (voltage, current, power), stores and analyzes it using a structured database, and provides insights through dashboards to optimize energy efficiency and detect anomalies.

Features
Real-time monitoring of machines
Data storage using MySQL
Interactive dashboard using Streamlit
Anomaly detection and forecasting
Technologies Used
Python (Streamlit)
MySQL
Pandas, NumPy, Plotly
SciPy
